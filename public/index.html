WHAT ARE WALLETS AND WHY DO I NEED A WALLET?

What are wallets and why do I need a wallet? Wallets allow you to keep track of your cryptocurrency holdings. The most important features of those wallets are the public and the private key. The public key is an identifier for your wallet. It works similar as an username on a social network. With the pubic key someone can send you money. The private key on the other hand is like your pin for your bank account.

BASIC RASPBERRY PI CONFIGURATION FOR HEADLESS MODE

Enable SSH Add an file to the boot partition called “ssh”. This enables you to connect to the pi via ssh. Enable WiFi To enable wifi on the raspberry pi add a file named “wpa_supplicant.conf” to the boot partition. Add following content to this file and adapt it to your needs. ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev country=<Insert 2 letter ISO 3166-1 country code here> update_config=1 network={ ssid="<Name of your wireless LAN>" psk="<Password for your wireless LAN>" }
